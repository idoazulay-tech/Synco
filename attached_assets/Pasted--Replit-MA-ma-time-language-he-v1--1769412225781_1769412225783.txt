אתה מפתח ב Replit את מנוע MA. תבנה מודול חדש בשם ma_time_language_he_v1 שמטרתו לזהות ולנרמל כל ביטויי הזמן בעברית לשפה פנימית אחידה.

הגדרת פלט אחיד (JSON):
- TimePoint: { type:"timepoint", date?: "YYYY-MM-DD", time:"HH:MM", timezone:"Asia/Jerusalem", confidence:0..1, reason?:string }
- DatePoint: { type:"datepoint", date:"YYYY-MM-DD", confidence, reason? }
- Duration: { type:"duration", duration_iso:"PnYnMnDTnHnMnS", confidence, reason? }
- Interval: { type:"interval", interval_iso:"<start>/<end or duration>", confidence, reason? }
- Recurrence: { type:"recurrence", rrule:string, start?: "<datetime>", confidence, reason? }

דרישות הבנה (עברית):
1) זמן מדובּר דיגיטלי:
   - "שמונה חמישים ותשע" => 8:59
   - "שמונה חמישים" => 8:50
   - שעה עגולה: "שמונה אפס אפס", "שמונה אפס", "שמונה בדיוק", "שמונה עגול", "שמונה על הדקה" => 8:00
   - תמיכה בדקות מ 00 עד 59, תמיד להחזיר דקות בשתי ספרות.

2) זמן בסגנון רבע/חצי:
   - "השעה אחת וחצי" => 1:30
   - "השעה שתיים ורבע" => 2:15
   - "רבע לשמונה" => 7:45
   - "שלוש חסר רבע" => 2:45
   - תמוך גם "עשרה לאחת" => 12:50 ו "חמישה לאחת" => 12:55 (אם מופיע).

3) חלקי יום:
   - בבוקר/בצהריים/אחר הצהריים/בערב/בלילה/לפנות בוקר
   - אם יש "בערב" תטפל כהמרה ל 24 שעות כשצריך (8 בערב => 20:00). אם חסר הקשר, תשאיר ותסמן confidence נמוך.

4) תאריכים יחסיים:
   - היום, מחר, מחרתיים, אתמול, שלשום
   - "בעוד X" / "תוך X" יחזיר Duration או Interval לפי ההקשר.

5) ימי שבוע:
   - יום ראשון..יום שבת וגם קיצורים יום א׳ וכו.
   - "ביום שני" => התאריך הבא של יום שני (עם timezone Asia/Jerusalem).

6) משכי זמן:
   - דקה/דקות, רבע שעה, חצי שעה, שעה/שעתיים, יום/יומיים, שבוע/שבועיים, חודש/חודשים
   - נרמל ל ISO 8601 duration: PT15M, PT30M, PT2H, P2D וכו.

7) טווחים:
   - "מ X עד Y", "בין X ל Y", "מחר מ X עד Y"
   - נרמל ל ISO 8601 interval <start>/<end>. אם חסר תאריך, אל תמציא, תשאיר Date חסר או תבקש clarification לפי מדיניות.

8) חזרתיות בסיסית:
   - "כל יום", "כל שבוע", "כל יום שני", "פעמיים בשבוע", "כל יומיים"
   - נרמל ל RRULE בסיסי כשאפשר. אם לא בטוח, החזר needs_clarification.

ארכיטקטורה חובה (ללא ספריות NLP כבדות):
- tokenizeHebrew(text): ניקוי טקסט, פירוק לטוקנים, טיפול בגרשיים/קיצורים.
- normalizeTokens(tokens): איחוד וריאציות ("ושלוש" => "ו","שלוש"), טיפול בא׳/ב׳ וכו.
- heNumberToInt(tokens): מילון מספרים + חיבור עשרות עם "ו".
- parseTimeHe(tokens): מחזיר timepoint חלקי (hour, minute, ampmHint).
- parseDateHe(tokens, now): מחזיר datepoint (כולל יחסי וימי שבוע).
- parseDurationHe(tokens): מחזיר duration_iso.
- parseIntervalHe(tokens, now): מחזיר interval_iso.
- parseRecurrenceHe(tokens, now): מחזיר rrule.
- main: parseTemporalHe(text, now, timezone="Asia/Jerusalem") שמחזיר אחד מחמשת הטייפים.

ולידציה:
- minutes חייב 0..59, hour חייב 0..23 כשיש הקשר 24 שעות.
- אם לא ניתן לפרש בביטחון, החזר {confidence נמוך, reason:"needs_clarification"}.
