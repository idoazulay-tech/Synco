# MA Mobile - אפליקציית המכולל

אפליקציית מובייל לניהול משימות עם MA (המכולל).

## התקנה והרצה

### דרישות
- Node.js 18+
- אפליקציית Expo Go על הטלפון (מחנות Play/App Store)

### הרצה מקומית
```bash
cd mobile
npm install
npm start
```

### שימוש
1. סרוק את ה-QR code עם Expo Go
2. הזן את כתובת השרת בהגדרות
3. התחל להשתמש!

## מסכים

### קלט
- הזנת טקסט חופשי
- הקלטה קולית (עובד רק על מכשיר אמיתי)
- שליחה ל-MA לניתוח

### שיקלול
- רשימת שאלות ממתינות מ-MA
- מענה כן/לא או טקסט חופשי
- עיצוב אחיד לכל בלוק שאלה

### הגדרות
- הגדרת כתובת השרת
- בדיקת חיבור

## חיבור לשרת
האפליקציה מתחברת ל-API הקיים:
- POST /api/analyze - ניתוח קלט
- POST /api/answer - מענה לשאלה
- GET /api/feedback - קבלת שאלות ממתינות
- POST /api/feedback/checkin/respond - מענה לצ'ק-אין
